# aa011666 — Personal Tools & Tweaks

This repository contains my personal collection of system-level tweaks, scripts, and utilities related to input latency, network optimization, and competitive gaming — mostly focused on Counter-Strike 1.6 and CS2.

I work with C++ and Python to build small tools that help improve responsiveness, reduce input lag, and make systems behave more predictably.  
Most of what’s here is based on real testing, spec documentation, and practical experience — not guesswork.

---

## 🧩 What’s Inside

- `src/cpp/` — C++ tools for diagnostics, registry patching, and low-level input handling
- `src/python/` — Python scripts for automation, config generation, and latency monitoring
- `reg/` — Registry tweaks for Intel/Realtek NICs and USB stack tuning
- `docs/` — Technical notes and bilingual guides (English / Ukrainian)

---

## 🧠 About Me

I'm interested in system optimization, especially for competitive games like CS 1.6 and CS2.  
I spend time researching how Windows handles input, interrupts, and scheduling — and how to tune it for better performance.  
I also write guides to help others apply these tweaks safely and effectively.
